Easy to use WordPress plugin to accept paypal payment for a service or product or donation in one click by shortcode

== Description ==
Easy to use WordPress plugin to accept paypal payment for a service or product or donation in one click. Can use it in sidebar, post and page of your website. It easy to use by shortcode.

The plugin will use shortcode.
[paypal-shortcode descriptions="This is description of paypal" email="email@your-email.com" options="Paypal for service 1: $45|Paypal for service 2: $55" buttom_image="the link image if has"]

== Installation ==
1. Unzip and Upload the folder 'sls-paypal-payments-shortcode' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings and configure the options eg. your email, Subject text etc.
4. See the usage section for details on how to place the paypal payment widget
